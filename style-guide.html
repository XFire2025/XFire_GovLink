Accessibility Standards

WCAG 2.1 AA Compliance

  • Color Contrast: Minimum 4.5:1 ratio for normal text, 3:1 for large text
  • Focus Indicators: Visible focus states on all interactive elements
  • Language Support: Multi-language content with proper lang attributes
  • Keyboard Navigation: All functionality accessible via keyboard
  • Screen Readers: Semantic HTML and proper ARIA labels

Focus States

focus:ring-2 focus:ring-primary/20 focus:border-primary

All interactive elements include visible focus indicators with the primary color and appropriate contrast.
